# TaskLog v3 — 気になってる事を置く場所

「タスク作っても結局やらない」自分のためのタスクアプリ。

- **置く** — 気になってる事を気軽に置く（やるとは言ってない）
- **触る** — ちょっとでも触れたら草（ヒートマップ）が生える
- **1分だけ** — 動けない日は AI が「最初の1分でやること」を3つ提案してくれる ← v3の新機能
- **手放す** — やらないと決めても咎められない

## v2 → v3 の差分

v2を作って気づいたこと：**タスクアプリを作った本人が、そのアプリを開かなかった。**
「置く」まではできても、置いた後に動き出すきっかけがなかった。

v3はそこに踏み込む：

1. **AI 1分ステップ分解** — 各タスクの「1分だけ」ボタンで、Claude（claude-haiku-4-5）が最初の1分で物理的に完了できる行動を3つ提案。選ぶと階段🪜としてタスクに付き、「やった」を押すと草が生える。APIキー未設定でも内蔵ロジックで動くフォールバック付き
2. **スマホ対応 / PWA** — ホーム画面に追加して1タップで開ける。スマホ入力時のズーム問題も修正
3. **小さいリワード** — 1分の前進にトーストで反応

## 技術

- Next.js (App Router) + Tailwind CSS、データはlocalStorage
- AI: Anthropic Messages API（structured outputs）。`ANTHROPIC_API_KEY` をVercelの環境変数に設定すると有効化

## 開発

```bash
npm install
npm run dev
```
