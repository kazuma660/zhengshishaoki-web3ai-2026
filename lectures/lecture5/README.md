# 第5回 プロトタイプ v1 — TaskLog

VPC v1（lecture3 参照）で構想したタスク管理プロダクトの動くプロトタイプ。

## 解決したいバグ
- タスク作っても結局やらない
- 思いついてもすぐ忘れる
- 何から始めるか分からない / やること多すぎ混乱

## 誰のためのプロダクト
毎日タスクをこなしたいが、優先順位がつけられず動けない人。

## 何が動くか
- タスク追加（タイトル + 重要度 + 期限）
- 重要度 × 期限の近さで**自動優先順位ソート**
- 1タップ完了
- **達成ログ可視化**：連続達成日数 / 今日の達成 / 累計 / 30日ヒートマップ
- localStorage で永続化（ログイン不要）

## スタック
Next.js 16 + TypeScript + Tailwind CSS + localStorage

## ローカル起動
```bash
cd task-mvp
npm install
npm run dev
```
