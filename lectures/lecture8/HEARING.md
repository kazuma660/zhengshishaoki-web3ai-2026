## 目的（このアプリで解こうとしてる問題）

「タスク管理が本当に自分の問題を解決するものなのか？」を考え直したら、違った。

自分の本当の問題はこれ：

```
学校・進路系（重い・抽象）を考えすぎる
  → 正解わからない／全部やらなきゃ／時間ない、のループ
  → 疲れる
  → 軽いタスク（英語・基本情報）にも手が出ない
  → 何もしない日
  → 数ヶ月積もって目標が遠ざかる
```

タスクを管理しても、このループの外に出るドアが要る。
v4 はそのドアを作ろうとしてる。

## 機能

- **置く / 明日やる の2ボタン** — 軽い思いつきは「明日やる」で摩擦ゼロで今日に入る
- **細かくする** — 抽象タスクを子タスクに分解（任意の深さ）。「TOEIC → 英単語20個」みたいに数えられる粒度に
- **▼ / ▶ 折り畳み** — 重いやつを視界から消せる（見ないといけないけど軽くしたい時用）
- **今日やる事** — 前夜セットしておけば朝は判断ゼロで動ける。「最初の一歩」を入力すると着火点が見える
- **やった** — 押して記録。何もしない日の対極を積み上げる
